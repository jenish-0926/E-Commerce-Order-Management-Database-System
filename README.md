
## Project Title
Zipgo E-Commerce Order Management System

## week 1

## Project Overview
This project is developed as part of the Database Management System (DBMS) course. The objective is to analyze the business requirements of an E-Commerce Order Management Database System and prepare the required documentation for database design.

## Week 1 Deliverables
- Requirement Analysis Report
- Business Requirement Document (BRD)
- Software Requirement Specification (SRS)
- README File

- # Week 2 – Entity and Relationship Analysis

## Description
This folder contains the Week 2 deliverables for the Amazon E-Commerce Order Management System project. The documents identify the system entities, their attributes, primary keys, foreign keys, relationships, and cardinalities required for designing the database.

## Files Included
- **Entity_Analysis_Report.pdf** – Details of all entities, attributes, primary keys, foreign keys, and constraints.
- **Entity_Relationship_Analysis.pdf** – Analysis of entity relationships, relationship types, and cardinality.

## Outcome
This analysis provides the database structure for the Zipgo E-Commerce Order Management System and serves as the foundation for creating the Entity Relationship (ER) Diagram and database implementation in the next phase.
